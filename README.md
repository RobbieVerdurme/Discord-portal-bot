## Discord protal bot
This is a portal bot that has some custom commands.

## Change
Look at [CHANGELOG](https://github.com/RobbieVerdurme/Discord-Portal-bot/blob/master/CHANGELOG.md)

## Build setup
``` bash
# install dependencies
npm install

# run node server
node index.js
```
